HTML Basico

O Que é HTML

Html é uma linguagem de marcação "Hypertext Markup Language" diferente das linguagens de progamação o Html é usado para definir uma estrutura do seu conteúdo "como um esqueleto " HTML consiste de uma serie de Elementos,que voçê usapra delimitar ou agrupar diferentes partes do seu conteudo determinado que ele apareça ou atue de tal maneira , As Tags anexas podem transformar uma palavra ou imagem num HiperLink, pode colocar as palavras em italico e podem aumentar e diminuir a fonte.

No geral ele define o que será mostrado ao cliente
